# Project Technologies

This project utilizes a modern full-stack architecture with a TypeScript-based backend and a React frontend.

## Backend

- **Language:** TypeScript
- **Runtime:** Node.js
- **Web Framework:** Express.js
- **ORM:** Prisma
- **Database:** PostgreSQL
- **Authentication:** better-auth
- **Package Manager:** npm

## Frontend

- **Language:** TypeScript
- **Framework:** React
- **Build Tool:** Vite
- **UI Libraries:** Radix UI, Shadcn/UI
- **Styling:** Tailwind CSS
- **Form Management:** React Hook Form
- **Schema Validation:** Zod
- **Routing:** React Router
- **Icons:** Lucide React
- **Notifications:** Sonner
- **Authentication:** better-auth
- **Package Manager:** npm
